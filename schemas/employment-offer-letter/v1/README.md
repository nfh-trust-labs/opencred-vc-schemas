# Employment Offer Letter Credential

**ID:** `employment-offer-letter/v1`
**Type:** `EmploymentOfferLetterCredential`
**Anchor standard:** Schema.org JobPosting / EmployeeRole; ISO 17442 LEI

## Description

Formal job offer extended by an employer to a candidate. Anchored on Schema.org JobPosting and EmployeeRole with vLEI-style employer identification.

## When to use

Use this credential when you need to issue a employment offer letter credential as a W3C Verifiable Credential. The schema is authored by OpenCred and anchored on Schema.org JobPosting / EmployeeRole; ISO 17442 LEI.
