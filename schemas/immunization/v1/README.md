# Immunization Credential

**ID:** `immunization/v1`
**Type:** `ImmunizationCredential`
**Anchor standard:** HL7 FHIR R4 Immunization; CVX/SNOMED/ATC code systems

## Description

W3C VC wrapper around HL7 FHIR R4 Immunization. Represents a single administered vaccine dose.

## When to use

Use this credential when you need to issue a immunization credential as a W3C Verifiable Credential. The schema is authored by OpenCred and anchored on HL7 FHIR R4 Immunization; CVX/SNOMED/ATC code systems.

## Schema URL

`https://raw.githubusercontent.com/nfh-trust-labs/opencred-vc-schemas/main/schemas/immunization/v1/schema.json`
