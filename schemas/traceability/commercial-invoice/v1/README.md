## Description

Commercial Invoice used in international trade — issued by a seller to a buyer and referenced by customs.

## When to use

Issuing a commercial invoice as a W3C Verifiable Credential in a supply-chain traceability context.

## Notes

This is a **referenced** credential: the authoritative schema lives upstream at the URL above.
