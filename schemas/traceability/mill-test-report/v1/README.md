## Description

Mill Test Report (MTR) — metallurgical certification document issued by a steel mill.

## When to use

Issuing a mill test report as a W3C Verifiable Credential in a supply-chain traceability context.

## Notes

This is a **referenced** credential: the authoritative schema lives upstream at the URL above.
