# Insurance Policy Credential

**ID:** `insurance-policy/v1`
**Type:** `InsurancePolicyCredential`
**Anchor standard:** Schema.org InsurancePolicy; ISO 4217 currency codes

## Description

Generic insurance policy credential with a policyType discriminator covering health, vehicle, life, travel, property, liability, and disability insurance.

## When to use

Use this credential when you need to issue a insurance policy credential as a W3C Verifiable Credential. The schema is authored by OpenCred and anchored on Schema.org InsurancePolicy; ISO 4217 currency codes.

## Schema URL

`https://raw.githubusercontent.com/nfh-trust-labs/opencred-vc-schemas/main/schemas/insurance-policy/v1/schema.json`
